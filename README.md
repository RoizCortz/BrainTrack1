# BrainTrack
A C# WinForms Project that is solely made for educational purposes to pass my Computer Programming Major. Purpose of this project is to make a Tracker for your educational needs, Tracks your assignment, activity, deadlines.  

## Features
 - Current
   - Login System using local access .mdb
   - Register System using local access .mdb
 - Planned
   - Tracking using Table format
   - Locally Saving without relying to .mdb
   - Export/Import your Tracks
